Graphs and Dashboards

  • Easily create dashboards with WYSIWYG drag-and-drop widgets
  • Build complex graphs to group and compare metrics an entire cloud
    service OR any amount of individual servers
  • Collect, correlate, compare, and contrast metrics and events from
    multiple systems
  • Gain deeper insight with composite metrics derived from
    mathematical formulas
 

Alerting

  • Alert on thresholds and rate of change for any metric collected by Datadog
  • Set alerts on an aggregated metric from multiple hosts OR individual
    devices/servers
  • Set alerts based on dynamically updating metadata tags, not individual hosts (unless you really want to)
  • Get context about any alert via email, SMS or through alerting apps
    like PagerDuty with contextual data (through notification method of
    your choice)
 

Collaboration

  • Collaborate on real-time events and outages with social-media style commenting
  • Reference historical conversations in full granularity
  • Snapshot, annotate, and discuss unusual time series
  • Share read-only dashboards via public URLs
  • Integrate Datadog activity into chat portals like Slack, HipChat, or Campfire
 

Built for Scale

  • Hosted solution allows for effortless monitoring scalability
  • Add as many hosts, team members and custom metrics as you have
  • How many servers/team members can we take on? All of them!
